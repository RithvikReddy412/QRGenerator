# QRGenerator

A web-based QR Code Generator built with HTML, CSS, and JavaScript that creates QR codes from user-provided text or URLs.

## Features

- Generate QR codes instantly
- User-friendly interface
- Responsive design
- Supports text and URLs

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.

## Future Improvements

- QR code download option
- Color customization
- Size adjustment
